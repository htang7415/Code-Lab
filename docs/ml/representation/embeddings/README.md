# Embeddings

Embeddings turn discrete or high-dimensional inputs into dense geometry that can be searched, clustered, and reused.

## Current Anchors

- Dense embeddings (`modules/ml/llm/embeddings`)
- Cosine similarity (`modules/ml/fundamentals/cosine-similarity`)
- Autoencoder compression (`modules/ml/representation/autoencoder`)
- Contrastive loss (`modules/ml/representation/contrastive-loss`)
- Pairwise ranking loss (`modules/ml/representation/pairwise-ranking-loss`)

## Concepts to Cover Well

- Embedding spaces as task-shaped geometry
- Dot product vs cosine similarity and the role of normalization
- Positive and negative pairs in contrastive training
- Collapse, anisotropy, and shortcut features
- Retrieval quality vs reconstruction quality
- Transfer learning, frozen embeddings, and probing
